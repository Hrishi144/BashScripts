# 📂 Count Files Bash Script

A lightweight, colorful Bash utility that counts all regular files in a given directory — with style and a dash of terminal flair.

---

## ✨ Features

✅ Counts all **regular files** in a directory  
✅ Displays **full paths** of each file  
✅ Shows an animated **progress bar**  
✅ Includes **color-coded** terminal output  
✅ Simple error handling for missing or invalid input  

---

## 🚀 Usage

```bash
./count_files.sh <directory_path>
