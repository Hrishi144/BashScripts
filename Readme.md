## 📁 Directory File Counter

A Bash script that:
- Accepts a directory path as an argument
- Lists all file paths inside it
- Shows the total number of files
- Adds cool terminal visuals (colors + animations)

---

## 📁 File Renamer

A Bash script that:
- Accepts a directory path, new name pattern, and extension as arguments
- Renames all files in the specified directory according to the new naming pattern
- Automatically increments the names of the files with each rename
